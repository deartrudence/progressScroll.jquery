progressScroll

The progressScroll jquery plugin

progressScroll adds a helpful and stylish horizontal scroll bar to the top of articles. Especially useful for long form articles, it allows the reader to understand how far they are through reading an article. Also helpful, progressScroll picks up all the tags along the way and presents them in the scroll bar.

progressScroll is customizable to fit in with the look and feel of your site. Everything from font family and size to the color and thickness of the scroll bar can be changed.

Adding progressScroll to your site

Simple

Simply add the jquery.progressScroll.js and jquery.progressScroll.css to your project. Remember to add a script tag for the .js file and a link tag for the .css file in your html and the latest version of jquery.

Once everything is set up all you have to do in your main javascript file is select the element for which the scroll bar will be used and append .progressScroll()

Then add the above html directly above the div you want the scroll bar for

With customization

If the default progressScroll doesn't quite fit with the look and feel of your site you can change that.

By adding any or all of the options available in the above picture you can change the color, font, width and whether the headings even appear at all.
